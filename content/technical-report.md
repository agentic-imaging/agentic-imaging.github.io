# Technical report (pending release gates)

The AgX technical report (arXiv) is being prepared against the same evidence discipline: every
quantitative claim in the report must resolve to an approved evidence record. The report covers
the system architecture, the two operating modes, the verifier and provenance subsystems,
matched-budget agent ablations, and an honest account of failure modes and limitations. It will
be linked here, with checksums, when the human release gate approves publication.[^3]

[^3]: Publication itself — the arXiv submission, code release, and this site's evidence pages
going live — is a human-executed transaction; the automation prepares it but cannot perform it.

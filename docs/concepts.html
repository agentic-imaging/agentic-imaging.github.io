    AgX is a loop of language-model agents that plan → run → verify → report a computational-imaging experiment, with skepticism built in. This page explains the moving parts.

    The loop

    • Plan. An agent drafts a plan with exact steps; adversarial reviewers critique it before anything runs.
    • Run. The plan is implemented and executed on the best available compute.
    • Verify. Independent verifiers check the code, the claims, and the physics of the measurement.
    • Report. Results are written with provenance-locked records, including negative and inconclusive ones.

    The anti-dishonesty apparatus

    • Measurement fidelity. Every reported metric carries a scale-invariant residual ‖A x̂ − y‖ / ‖y‖; results inconsistent with the physics are flagged. See Reproducibility.
    • Condition schemas. Every result declares a condition_schema_id; a bare label is invalid and no comparison crosses schemas.
    • Human gates. A claim is publishable only after human approval; publication is a human transaction.

    Where the data lives

    Runs emit an on-disk trace (the canonical record the Studio visualizes). Its contract is frozen; see Reference. To reproduce a number end to end, see Reproducibility.
